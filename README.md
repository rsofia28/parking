# Parking Lot Management System

> Description

This project is a simple parking management system written in Python.  
It is a program that simulates a parking lot with ten available spaces.  
The system allows users to register cars using their license plates.

The program is designed to help users manage parking spaces easily.

## How it works

1. The program creates a list with ten empty parking spaces.
2. The system shows a menu with four options: show spaces, enter a car, remove a car, and exit.
3. The user selects an option and the program processes the request.
4. When a car enters, the program stores the license plate in the first available space.
5. When a car leaves, the program removes the plate and frees the space.
6. If the parking lot is full, the system displays a warning message.
7. The program continues running until the user chooses the exit option.

## Status

The system is currently running in a continuous loop and is waiting for user interaction.
